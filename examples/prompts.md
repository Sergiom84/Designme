# Prompts De Ejemplo

Usa estos prompts para revisar regresiones visuales, calidad del brief y cobertura de artefactos.

## Software

```txt
Software para operaciones de soporte B2B con cola de tickets, SLA, responsables, escalados y panel de acciones rápidas.
```

## Web

```txt
Web de producto para una herramienta de IA que convierte reuniones en tareas verificables, con prueba clara, flujo de trabajo y solicitud de demo.
```

## Dashboard

```txt
Dashboard para un CRM de ventas B2B con pipeline, riesgos, deals bloqueados, forecast y siguientes acciones por owner.
```

## Mobile

```txt
App móvil de hábitos para fundadores ocupados con foco diario, rachas, energía, captura rápida y revisión semanal.
```

## Deck

```txt
Deck de lanzamiento para explicar una plataforma local-first de diseño con agentes, export HTML, crítica y privacidad.
```

## Infografía

```txt
Infografía sobre cómo reducir deuda técnica en equipos pequeños de producto, con señales, coste, prioridad e impacto.
```

## Prompt De Estrés

```txt
Dashboard ejecutivo para una fintech: ingresos, fraude, cohortes, alertas de cumplimiento, decisiones pendientes y plan de respuesta.
```

## Qué Mirar

- El dominio detectado debe aparecer en el inspector de direcciones.
- La crítica debe listar incidencias accionables, no opiniones genéricas.
- El HTML exportado debe mantener `<!doctype html>`, viewport y metadatos UX.
- Los modos escritorio, tablet y móvil no deben romper la lectura.
- El handoff debe tener suficiente contexto para continuar fuera de Designme.
